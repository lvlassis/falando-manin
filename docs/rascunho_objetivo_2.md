# Qual a sintaxe da linguagem?

Uso de sinônimos para os mesmos comandos.

## ESTRUTURA

ESTRUTURA
- LINGUAGEM


Referência pseudocódigo
```
SIGNIFICADO
```

```
SINTAXE DA LINGUAGEM
```


Exemplo de uso
```
EXEMPLO COM A LINGUAGEM
```


## print

print:
- "gritarei"
- "digo que"

```
print("meu amor")
print("nunca te abandonarei")
```

```
gritarei meu amor
digo que nunca te abandonarei
```


## if-else

if:
- "se"

else if:
- "porém, se"
- "porém, caso"

else:
- "senão"

end:
- "." (ponto final)


Referência:
```
if ( amor != forte ) {
  print("partirei")
} else {
  print("estarei sempre aqui")
}
```

Sintaxe:
```
se x (condição) então
  { código }
porém, se|caso (condição) então
  { código }
senão
  { código }.
```

Exemplo de uso:
```
se o amor não for forte então
grito que partirei
porém, se o coração arder então
digo que posso continuar
senão, digo que não vou mais lhe incomodar.
```



## Laços de repetição

for:
- "para"

while:
- "enquanto"

end:
- "." (ponto final)

```
while ( amor == "durar" ) {
  print("minha lealdade será sua")
}
```

```
enquanto x (condição) 
{ código }.
```

```
enquanto o amor durar
digo que minha lealdade será sua.
```



## Atribuição

=
- "é"
- "és"
- "me lembra"
- "me lembram"


Referência pseudocódigo
```
amor = "fogo que arde sem se ver"
amor = "bossa nova"
sexo = "carnaval"
```

Sintaxe
```
(variável) é|és|me lembra (valor)
```


Exemplo de uso
```
amor é fogo que arde sem se ver
amor é bossa nova
sexo é carnaval
```
